# 贪心算法总结

#### LC55.跳跃游戏

```python
class Solution:
    def canJump(self, nums: List[int]) -> bool:
        n = len(nums)
        farthest = 0
        for i in range(n - 1):
            farthest = max(farthest, i + nums[i])
            if farthest <= i: return False
        return farthest >= n - 1
```
