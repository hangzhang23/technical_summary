# 总结二分查找框架和题目
## 关键词：有序数组，寻找一个数，寻找左边界，寻找右边界。  
例题：递增数列找数, 有数组和目标数，有则返回目标数下标，没有则返回-1。
```python
def binary_search(nums, target):
    left, right = 0, len(nums)-1
    while left <= right:
        mid = (left + right) // 2
        if target == nums[mid]: return mid
        elif nums[mid] > target:
            right = mid - 1
        else:
            left = mid + 1
    return -1
```
